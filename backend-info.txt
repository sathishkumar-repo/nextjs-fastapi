Install PostgreSQL in DOCKER

1. docker pull postgres
2. docker run -d --name fastapi-postgres -p 5432:5432 -e POSTGRES_USER=postgres -e POSTGRES_PASSWORD=postgres -e POSTGRES_DB=fastapi_db postgres
3. Verify container => docker ps
4. Verify Database Inside Docker => docker exec -it fastapi-postgres psql -U postgres -d fastapi_db
    Show tables => \dt
    Show rows => SELECT * FROM users;

====================================================
Postgres UI

1. Run pgAdmin in Docker => docker run -d --name fastapi-pgadmin -p 5050:80 -e PGADMIN_DEFAULT_EMAIL=admin@admin.com -e PGADMIN_DEFAULT_PASSWORD=admin dpage/pgadmin4
2. Open browser => http://localhost:5050
3. Login:
    Email: admin@admin.com
    Password: admin
4. Add PostgreSQL Server
    Name: FastAPI PostgreSQL
    Connection Tab:
        Host name/address: host.docker.internal
        Port: 5432
        Username: postgres
        Password: postgres
        Database: fastapi_db        
5. Explore Database
    Servers
     └ PostgreSQL
        └ Databases
           └ fastapi_db
              └ Schemas
                 └ public
                    └ Tables
